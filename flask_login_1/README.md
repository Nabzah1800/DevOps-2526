# flask_login_1

Deze map bevat een Flask applicatie met login en registratie functionaliteit.

## Bestanden
- **app.py**: Hoofdapplicatiebestand met login en registratie routes.
- **db.py**: Database interactie en gebruikersbeheer.
- **Dockerfile**: Docker configuratie voor de app.
- **requirements.txt**: Benodigde Python packages.
- **static/style.css**: Stijlen voor de webpagina's.
- **templates/home.html, login.html, register.html**: HTML templates voor de verschillende pagina's.

Gebruik deze app als basis voor authenticatie in Flask.
