# DevOps-2526

Dit project bevat verschillende labs en demo's voor DevOps, waaronder Ansible, Flask-applicaties, Python scripts en unit testing.

## Structuur

- **ansible_apache_lab/**: Bevat Ansible configuratie voor het opzetten van een Apache server.
- **flask_app_demo/**: Eenvoudige Flask demo-app met statische bestanden.
- **flask_login_1/**: Flask app met login functionaliteit en database.
- **flask_login_2/**: Uitgebreidere Flask login app, inclusief Jenkinsfile voor CI/CD.
- **python_script/**: Python script voor het toevoegen van 100 willekeurige boeken.
- **unit_testing/**: Voorbeeld van unit testing in Python.

Elke submap bevat een eigen README met meer details.
