# flask_login_2

Deze map bevat een uitgebreidere Flask login applicatie met CI/CD integratie.

## Bestanden
- **app.py**: Hoofdapplicatiebestand met login, registratie en home routes.
- **db.py**: Database interactie en gebruikersbeheer.
- **Dockerfile**: Docker configuratie voor de app.
- **Jenkinsfile**: CI/CD pipeline configuratie voor Jenkins.
- **requirements.txt**: Benodigde Python packages.
- **static/style.css**: Stijlen voor de webpagina's.
- **templates/home.html, login.html, register.html**: HTML templates voor de verschillende pagina's.

Deze app is geschikt om te oefenen met authenticatie, database en CI/CD.
