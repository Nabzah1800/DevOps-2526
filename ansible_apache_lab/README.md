# ansible_apache_lab

Deze map bevat een Ansible lab voor het installeren en configureren van een Apache webserver.

## Bestanden
- **ansible.cfg**: Ansible configuratiebestand.
- **apache.yml**: Playbook voor het installeren en configureren van Apache.
- **configure_ssh_server.sh**: Script om SSH server te configureren.
- **hosts**: Inventarisbestand met de doelhosts.
- **templates/index.html**: Template voor de Apache webpagina.

Gebruik dit lab om te oefenen met Ansible en het automatiseren van serverconfiguratie.
