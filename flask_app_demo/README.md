# flask_app_demo

Deze map bevat een eenvoudige Flask demo-applicatie.

## Bestanden
- **app.py**: Hoofdapplicatiebestand voor Flask.
- **Dockerfile**: Docker configuratie om de app te containeriseren.
- **requirements.txt**: Benodigde Python packages.
- **static/style.css**: Stijlen voor de webpagina.
- **templates/index.html**: HTML template voor de homepage.

Gebruik deze demo om te leren werken met Flask en Docker.
